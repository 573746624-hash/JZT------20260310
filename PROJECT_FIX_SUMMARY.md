### **问题2: ES模块路径别名配置**
```typescript
// ❌ 错误（在ES模块中__dirname不可用）
import path from "path";
alias: {
  '@': path.resolve(__dirname, './src')
}

// ✅ 修复
import { fileURLToPath, URL } from 'node:url';
alias: {
  '@': fileURLToPath(new URL('./src', import.meta.url))
}
```
**影响**: 导致构建失败  
**状态**: ✅ 已修复
