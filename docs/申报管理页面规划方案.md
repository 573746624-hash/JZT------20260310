# 申报管理页面全面重新规划方案

> 版本：v1.0  
> 创建时间：2026-03-23  
> 依据：申报管理模块需求方案 + 现有系统分析

---

## 一、现有系统分析

### 1.1 现有页面结构

```
src/pages/application/
├── index.tsx                    # 申报管理主页面（当前）
├── ApplyWizardWithLayout.tsx    # 申报向导页面（当前）
├── MyApplications.tsx           # 我的申报入口
├── OptimizedMyApplications.tsx  # 优化版我的申报
├── ApplySuccess.tsx             # 申报成功页
├── PolicyDetail.tsx             # 政策详情页
└── components/
    ├── QualificationDrawer.tsx  # 资质选择抽屉
    └── QualificationSelector.tsx # 资质选择器
```

### 1.2 当前存在的问题

1. **页面职责不清晰**：`index.tsx` 混合了政策浏览和申报管理功能
2. **缺少企业级仪表盘**：没有从管理视角的数据概览
3. **向导流程孤立**：申报向导与列表页面缺乏有效关联
4. **状态展示不直观**：申报进度和状态展示不够清晰
5. **操作流程不连贯**：从查看到申报的流程不够顺畅

### 1.3 现有申报流程梳理

```
现有流程：
政策中心 → 政策详情 → 立即申报 → 申报向导（4步骤）→ 申报成功
                    ↓
              我的申报列表（查看进度）
```

## 三、页面详细规划

### 3.1 申报管理仪表盘（企业级）

**页面路径**：`/application/management`

**页面定位**：企业申报事务的统一管理入口，提供数据概览和快捷操作

#### 3.1.2 核心功能模块

**1. 统计卡片区**
- 展示5个核心统计指标
- 点击卡片可下钻到对应筛选状态的列表
- 数字变化动画效果
- 环比变化指示（上升/下降）

**2. 申报趋势图表**
- 使用ECharts折线图展示近6个月申报量
- 区分：新建数、通过数、驳回数
- 支持时间范围切换（近3月/近6月/近1年）

**3. 快捷操作区**
- 新建申报：启动申报向导
- 导入草稿：支持从本地导入草稿文件
- 查看报表：跳转到统计分析页
- 归档管理：查看已归档申报

**4. 待办提醒区**
- 即将截止提醒（3天内）
- 需补充材料提醒
- 审核反馈提醒
- 点击跳转对应列表

**5. 最近申报项目**
- 展示最近更新的5条申报
- 显示项目名称、当前步骤、进度条、截止日
- 快捷操作按钮（继续/查看）
- 点击查看全部进入列表页

#### 3.2.2 核心功能模块

**1. 筛选栏**
- 政策类型：多选下拉框（高企认定、专精特新、研发加计等）
- 申报状态：多选下拉框（草稿、填写中、待提交、审核中等）
- 时间范围：日期范围选择器（创建时间/截止时间）
- 负责人：人员选择器
- 搜索框：支持项目名称、政策编号模糊搜索
- 清除筛选：一键重置所有筛选条件

**2. 批量操作栏**
- 全选/取消全选
- 批量导出：导出Excel报表
- 批量提醒：发送站内信提醒
- 批量归档：将已完成申报归档
- 视图切换：列表视图/卡片视图

**3. 申报列表**
- 表格字段：项目名称、当前步骤、状态标签、负责人、截止日
- 行操作：查看、编辑、删除、指派、提醒
- 行内编辑：支持快捷修改负责人
- 排序：支持多字段排序

**4. 分页器**
- 支持页码跳转
- 每页条数设置（10/20/50/100）
- 显示总条数
